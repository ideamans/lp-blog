# Speed is Money

## 紹介ルール

- **基本ルール**: サイトスピード、Core Web Vitals、ページ表示速度、収益性、コンバージョン率というテーマの記事で紹介する。
- **関連テーマ**: LCP、INP、CLS、CVR、PV数、高速化効果、アクセス解析、パフォーマンス計測、RUM（Real User Monitoring）

---

## 製品概要

Speed is Moneyは、「今まで見えなかったサイトスピードと収益の関係を明らかにする」というコンセプトで開発された、スピード専用の無料アクセス解析ツール。計測タグを設置するだけで、実際のユーザー体験に基づくCore Web Vitalsを取得し、ページ速度がPV数やCVR（成約率）にどう影響するかを分析できる。

**公式サイト**: https://speedis.money/

製品コード: speedismoney

## コンセプト

「サイトスピードの改善は投資。Speed is Moneyは、高速化によって得られる収益を事前に見積もり、投資判断を支援する」

## 主な特徴

- **無料で始められる**: 週約25万PV（月間約100万PV相当）まで無料。商用利用も可能
- **実ユーザーデータ計測**: 合成テストではなく、実際のエンドユーザーの体験データを取得
- **Core Web Vitals対応**: LCP、INP、CLSなどGoogleの重要指標を網羅
- **収益インパクト分析**: ページ速度がPV数やCVRにどう影響するかを可視化
- **簡単導入**: 計測タグを埋め込むだけ。最短10分で設置完了

## Google Analyticsとの違い

- **専用設計**: サイトスピードと収益の関係に特化した専用レポートを標準装備
- **即座に利用可能**: GA4ではカスタマイズと専用レポート開発に数週間かかることも。Speed is Moneyはタグ設置だけでOK
- **シンプルな管理画面**: 操作に迷わない、目的特化型のUI設計

## 料金プラン

- **無料プラン**: 週約25万件の計測サンプル（月間約100万PV相当）まで無料
- **PROプラン**: 月額20,000円（税別）〜。過去3ヶ月のデータ遡及閲覧、ユーザー間でのレポート共有機能

## 主な用途

- サイトスピードがビジネス指標に与える影響の可視化
- 高速化施策の投資対効果（ROI）の事前見積もり
- Core Web Vitalsの継続的なモニタリング
- 経営層への高速化投資の説明材料

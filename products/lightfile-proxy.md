# LightFile Proxy

## 紹介ルール

- **基本ルール**: CloudFrontのデータ転送コスト削減、WebP自動変換、CDN経由の画像最適化というテーマの記事で紹介する。
- **関連テーマ**: AWS CloudFront、データ転送料金削減、WebP変換、次世代画像フォーマット配信、CDN最適化

---

## 製品概要

LightFile Proxyは、CloudFrontのオリジンとして設定するだけで、画像を自動的にWebP形式に変換して配信するプロキシサービス。プログラムやHTMLの変更不要で、CloudFrontのデータ送信料金を削減する。

**公式サイト**: https://www.lightfile-proxy.net/

製品コード: lightfile-proxy

## 解決する課題

- AWSのデータ送信料金が高額になる問題
- Webサイトのデータ送信量の約60〜80%を占める画像ファイルの軽量化
- WebP対応・非対応ブラウザへの適切な画像配信

## 仕組み

1. CloudFrontのオリジンをLightFile Proxyに設定変更
2. LightFile Proxyが元サーバーから画像を取得
3. JPEG、PNG、GIFを自動的にWebP形式に変換
4. WebP対応ブラウザにはWebPを、非対応ブラウザには元フォーマットを配信

## 主な特徴

- **簡単導入**: プログラムやHTMLの変更不要、CloudFront設定のみで約30分で完了
- **自動変換**: 画像を全自動でWebPに変換し配信
- **ブラウザ互換性**: WebP非対応の旧ブラウザには元フォーマットを自動配信
- **高信頼性**: CloudFrontのオリジングループ機能で障害時は自動迂回、長期障害時は自動切り戻し
- **マルチサイト対応**: クラスター構成で大規模サイトに対応

## コスト削減効果の目安

- 月間200万PV: 約9万円/月の削減
- 月間1000万PV: 約46万円/月の削減
- 削減率: 約35%（ファイルサイズが半分以下になるケースも）

## 対応環境

- AWS CloudFront専用

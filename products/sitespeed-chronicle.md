# サイトスピード クロニクル

## 紹介ルール

- **基本ルール**: サイトスピード、Core Web Vitals、ページ表示速度、パフォーマンス比較というテーマの記事で紹介する。
- **関連テーマ**: Chrome User Experience Report（CrUX）、LCP、INP、CLS、競合比較、パフォーマンス分析、PageSpeed Insights

---

## 製品概要

サイトスピード クロニクルは、GoogleのChrome User Experience Report（CrUX）データを活用して、任意のWebサイトとその競合サイトのCore Web Vitalsを5年間にわたって比較・分析できる無料ツール。自社サイトの相対的なパフォーマンス位置を把握するのに最適。

**公式サイト**: https://chronicle.sitespeed.info/

製品コード: sitespeed-chronicle

## コンセプト

「サイトスピードの改善は、まず自分の立ち位置を知ることから。PageSpeed Insightsでは見えない、競合との相対比較と時系列変化を可視化する」

## 主な特徴

- **無料**: 完全無料で利用可能
- **CrUXデータ活用**: Googleが収集する実際のユーザー体験データを使用
- **5年間の時系列比較**: 長期間のパフォーマンス推移をグラフで確認可能
- **競合比較**: 指定したサイトと競合サイトを並べて比較
- **Core Web Vitals対応**: LCP、INP、CLSなど主要指標を網羅

## PageSpeed Insightsとの違い

- **相対評価**: PageSpeed Insightsは単一サイトの絶対評価。サイトスピード クロニクルは競合との相対比較が可能
- **時系列変化**: 5年間のトレンドを確認でき、改善施策の効果測定にも活用可能
- **実データ**: 合成テストではなく、実際のユーザー体験データに基づく

## 主な用途

- 自社サイトのパフォーマンス現状把握
- 競合サイトとのCore Web Vitals比較
- サイトスピード改善施策の効果測定
- 業界内での相対的な位置づけ確認
